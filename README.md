## Requerimentos

* NodeJS com NPM: [link](https://nodejs.org/)

## Instalação

* Clone o repositório `git clone https://github.com/RobertoMaioli/Angular-Project.git`
* Instale os pacotes npm `npm install`

### Rodando localmente

* `gulp serve`
* Aponte o navegador para `localhost:3000`
